# Kalkulator Zakat - README

## Deskripsi

Website **Kalkulator Zakat** interaktif berdasarkan Tabel Nishab BAZNAS Kota Semarang. Dibangun dengan HTML, CSS (Bootstrap 5), dan vanilla JavaScript.

## File Structure

```
zakat-multi/
├── index.html              # Main HTML file
├── styles.css              # Custom CSS styling
├── script.js               # JavaScript calculator logic
├── tabel_nishab_baznas.jpg # Referensi tabel BAZNAS
├── WALKTHROUGH.md          # Dokumentasi lengkap
└── README.md               # File ini
```

## Fitur

### 5 Jenis Kalkulator Zakat:

1. **Zakat Fitrah** - 3.5 kg beras per jiwa
2. **Fidyah Puasa** - 1.5 kg beras per hari
3. **Zakat Mal** - Emas, Uang, Usaha (Nishab: 85g emas, Kadar: 2.5%)
4. **Zakat Pertanian** - Beras (653kg) & Palawija (815kg), Kadar: 5-10%
5. **Zakat Modern** - Profesi, Saham, Peternakan (Kadar: 2.5%)

### Fitur Teknis:

✅ Responsive design (mobile & desktop)

✅ Bootstrap 5 UI components

✅ Tab navigation

✅ Form validation

✅ Nishab validation otomatis

✅ Format currency Rupiah

✅ Smooth animations

✅ Print-friendly

## Cara Menggunakan

1. Buka file `index.html` di browser
2. Pilih jenis zakat yang ingin dihitung
3. Isi form dengan data yang diperlukan
4. Klik tombol "Hitung"
5. Lihat hasil perhitungan
